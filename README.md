## About / 关于我

**中文**

我是 **Tutict White**，GitHub ID 是 [`tutict`](https://github.com/tutict)。我关注把想法推进成可运行、可验证、可维护的系统：从 AI 工作流、金融研究智能体、本地桌面 Agent 工作台，到 Flutter 移动端、Java/Quarkus 后端和自动化工具链。

近期公开项目更集中在三类方向：AI 工程交付与 Agent 编排、本地优先的桌面/协作工具，以及面向业务场景的全栈产品系统。

**English**

I am **Tutict White**, known as [`tutict`](https://github.com/tutict) on GitHub. I build practical systems that turn ideas into runnable, testable, and maintainable software: AI workflow platforms, financial research agents, local-first desktop agent workbenches, Flutter clients, Java/Quarkus services, and automation tooling.
